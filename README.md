# 🛒 Green Basket System

A simple Java-based inventory management system built using Object-Oriented Programming concepts.

## 🚀 Features
- User login simulation
- Inventory management
- Product storage using ArrayList
- File handling (load data from file)
- Display product details (price, quantity)

## 🧠 Concepts Used
- Java OOP (Classes, Objects, Methods)
- Collections (ArrayList)
- File Handling

## 📌 Sample Output
Apple | Fresh Produce | Rs.150.00 | Qty: 20  
Milk | Dairy | Rs.250.00 | Qty: 5
